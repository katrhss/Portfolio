Welcome

I'm Katris Konstantinos

I'm 20 years old, studying Computer Science at the University of Western Macedonia (UoWM) at Kastoria, Greece. My programming career started around the time I got into University. Since then, I've seen numerous fields, as well as programming languages, throughout my semesters, but Web Development is what attracted me, with JavaScript intriguing me the most. I've seen mostly Vanilla JavaScript along with HTML and CSS. In addition, I've worked with a little bit of React, some React-Native for Mobile App Development, as well as Node.js for some Backend Server Applications.
